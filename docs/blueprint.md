# **App Name**: BloomWell

## Core Features:

- Personalized Daily Tips: Generate 5-10 daily health and wellness tips based on onboarding preferences and activity levels. A generative AI tool refines tip selection based on user progress and long-term goals.
- Interactive Calendar: Allow users to schedule and track personal health activities, like yoga sessions, runs, or meditation, with custom events, reminders, and notifications.
- Activity Hub: Provide curated lists of local resources such as sports locations, relaxation activities, healthy stores, outdoor spaces, healthy recipes and local experts with contact options.
- Progress Tracking: Visualize user progress through a horizontal progress bar and track stats like daily streak, tips completed, and activities logged. Display the gathered information as bar or circular charts.
- Social Feed Simulation: Enable users to view simulated social profiles with recent activities and common interests to foster a sense of community and motivation, with dummy match function for increased engagement.
- Profile Management: Enable users to manage personal information, edit preferences, track activity history, and adjust app settings such as notifications and login information.
- Onboarding Experience: Guide new users through a three-screen onboarding process to gather initial preferences (motivations, favorite activities, time commitment) and personal information (name, email, location, birthdate, profile picture).

## Style Guidelines:

- Primary color: Turquoise (#17B5B5) to evoke a sense of health and well-being.
- Background color: Light turquoise (#C5F0F0), a desaturated tone from the primary hue, creating a calm background.
- Accent color: Light teal (#6CD7D7), an analogous hue that offers subtle contrast and draws the eye to CTAs.
- Body and headline font: 'Poppins' for a contemporary, precise feel.
- Consistent and clear icons from Expo Vector Icons to represent activities, categories, and actions throughout the app.
- Modern, youthful, and minimalist design using grids and cards to display tips, activities, and other content in an organized manner.
- Subtle animations and transitions for UI elements to provide engaging feedback and enhance user experience.
